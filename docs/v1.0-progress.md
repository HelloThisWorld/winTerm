# Historical winTerm v1.0 Progress

This file records the completed v1.0.2 MSIX-era release work. It is retained for
historical traceability and is not the current release plan.

The active distribution design uses an unpackaged, self-contained runtime with
an Inno Setup installer and a Portable ZIP. See:

- `docs/current-progress.md` for the current implementation and test status;
- `docs/release-process.md` for the active release gates;
- `docs/user/installation.md` for supported installation methods.

Existing v1.0.x tags and GitHub Releases are immutable historical releases. Do
not replace their assets or reuse their tags.

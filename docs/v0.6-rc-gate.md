# winTerm v0.6 release candidate gate

The gate is deliberately incomplete until its stated evidence exists.

- [ ] No known P0 issues remain.
- [ ] P1 issues are fixed or explicitly accepted in the release report.
- [x] Public Beta feature classification is documented.
- [x] Privacy and security policies are present.
- [x] Source and fixture validation is automated in CI.
- [ ] Windows 11 x64 clean install, upgrade, downgrade, and uninstall are tested.
- [ ] ARM64 package is built and basic-launched, or ARM64 remains explicitly unsupported.
- [ ] Workspace recovery and docking rollback receive compiled runtime verification.
- [ ] Diagnostic redaction tests pass.
- [ ] SBOM and release checksums are generated from release artifacts.
- [ ] Release notes and known issues are reviewed.
- [ ] WinGet manifest validates with final release URLs and hashes.

The current Public Beta decision is **Not ready for public publication**. This branch prepares the candidate infrastructure; it does not manufacture missing runtime evidence.

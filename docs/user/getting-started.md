# Getting started

Launch winTerm, choose an available shell profile, and open a tab.

To split the focused pane, open the pane menu or Command Palette, choose
**Split Pane**, and select **Top**, **Bottom**, **Left**, or **Right**. The new
profile opens on the selected side of the focused pane. Profile selection,
default/current profile behavior, minimum dimensions, and failed-shell
rollback remain part of the normal split flow.

In a split layout, compact pane headers show a pane icon, title, status, and
overflow button. Click the icon to focus that pane. The header does not move
the pane. Use the overflow button or right-click the header for the same pane
menu.

Resize sibling panes by dragging their shared border. The border snaps to
quarters, thirds, and 50/50 by default. Hold **Alt** while dragging for free
resizing, press **Escape** to restore the original ratio, or use **Balance
Panes** to return the immediate split to equal size.

Final split ratios and nested structure are restored with persisted layouts.
See [Visual layout](visual-layout.md), [Keyboard shortcuts](keyboard-shortcuts.md),
and [Settings](settings.md).

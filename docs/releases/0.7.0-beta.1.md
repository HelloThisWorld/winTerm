# winTerm 0.7.0-beta.1

## Download

Download one of these x64 distributions from the official winTerm
`v0.7.0-beta.1` GitHub Release:

- `winTerm-0.7.0-beta.1-setup-x64.exe` — the recommended Windows 11 installer;
- `winTerm-0.7.0-beta.1-portable-x64.zip` — the portable build, with settings
  stored beside the application in `data`;
- `SHA256SUMS.txt` — SHA-256 hashes for every published asset.

The release also includes third-party notices, SPDX and CycloneDX SBOMs, and
the release notes. The application version is `0.7.0-beta.1`.

## Important preview and signing notice

This is a Beta preview. The Setup EXE in this release is **unsigned** because a
trusted Authenticode certificate is not configured. Windows SmartScreen may
show an unknown-publisher warning. Download only from
`github.com/HelloThisWorld/winTerm` and verify `SHA256SUMS.txt` before running
it. No certificate import or MSIX sideloading is required.

## What changed

- Added Top, Bottom, Left, and Right directed splits relative to the focused
  pane, with rollback when profile or pane creation fails.
- Added pane headers, drag handles, unified pane menus, docking previews, move
  actions, keyboard targeting, and accessible announcements.
- Added an unpackaged, self-contained runtime with Inno Setup and Portable ZIP
  distributions.
- Added installer registration, upgrade/reinstall, data-preservation, layout,
  checksum, SBOM, and release-workflow validation.

## Supported platform

Windows 11 x64 is the supported preview target. ARM64 and Windows 10 are not
part of this release.

## Installation and isolation

The installer registers the `winterm.exe` command and does not register or
replace Windows Terminal's `wt.exe`. Because Windows filenames are
case-insensitive, the product-cased `winTerm.exe` is also directly invokable as
`winterm.exe`.

Portable users should extract the entire ZIP to a writable directory and run
`winTerm.exe`. Keep the `portable.marker` file in place to retain portable
settings behavior.

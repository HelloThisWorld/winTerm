# winTerm 0.7.0-beta.1

## Download

Download these files from the official winTerm `v0.7.0-beta.1` GitHub Release:

- `winTerm-0.7.0-beta.1-x64.msix` — the Windows 11 x64 preview installer;
- `winTerm-0.7.0-beta.1.cer` — the public self-signed code-signing certificate;
- `INSTALL.txt` — certificate trust, installation, and removal instructions;
- `SHA256SUMS.txt` — SHA-256 hashes for the published assets.

The application display version is `0.7.0-beta.1`. The Windows/MSIX package
version is `1.0.3.0` so this preview can upgrade an existing `1.0.2.0`
installation.

## Important preview and signing notice

This is a Beta preview. Runtime and manual acceptance remain incomplete; see
`docs/v0.7-acceptance.md` for the exact open evidence.

The installer is cryptographically self-signed with
`CN=winTerm Development`; it is not signed by a public certificate authority.
Windows will not trust it automatically. Verify that every file came from
`github.com/HelloThisWorld/winTerm`, check the SHA-256 hashes, and import only
the certificate attached to the same Release.

## What changed

- Added Top, Bottom, Left, and Right directed split planning relative to the
  focused pane.
- Added compact pane-header, drag-handle, pane-menu, and Snap-layout
  presentation models.
- Added transaction-ready pane movement plans that retain pane and session
  identity.
- Added keyboard pane-move targeting, accessible announcements, and
  configurable pane-control settings.
- Added automated coverage for directed split and pane-control foundations.

## Supported platform

Windows 11 x64 is the supported preview target. ARM64 and Windows 10 are not
part of this Release.

## Installation

Follow `INSTALL.txt` from the same Release. The package registers
`winterm.exe`; it does not register or replace Windows Terminal's `wt.exe`.

# winTerm 0.7.0-beta.4

## Download

Download one of these x64 distributions from the official winTerm
`v0.7.0-beta.4` GitHub Release:

- `winTerm-0.7.0-beta.4-setup-x64.exe` — the recommended Windows 11 installer;
- `winTerm-0.7.0-beta.4-portable-x64.zip` — the portable build, with settings
  stored beside the application in `data`;
- `SHA256SUMS.txt` — SHA-256 hashes for every published asset.

The release also includes third-party notices, SPDX and CycloneDX SBOMs, and
these release notes.

## Important preview and signing notice

This release is currently unsigned.

Windows may display an Unknown Publisher or Microsoft Defender SmartScreen
warning. Verify the downloaded file using `SHA256SUMS.txt` before running the
installer. No certificate import, Developer Mode, or MSIX sideloading is
required.

## What changed

- Added the unpackaged, self-contained Inno Setup installer as the recommended
  distribution.
- Added a Portable ZIP with adjacent `portable.marker` and `data` storage.
- Unified winTerm-owned Company, Product, About, installer, update, SBOM, and
  release metadata under `helloThisWorld`.
- Added strict branding, layout, checksum, installation, launch, upgrade,
  uninstall, and public re-download verification.
- Includes directed Top, Bottom, Left, and Right splits, pane headers, drag
  handles, Snap-like previews, unified pane menus, and same-process pane moves.

## Supported platform

Windows 11 x64 is the supported preview target. ARM64 and Windows 10 are not
part of this release.

## Installation and isolation

The installer registers the `winterm.exe` command and does not register or
replace Windows Terminal's `wt.exe`. It preserves `%LOCALAPPDATA%\winTerm` on
uninstall unless the user explicitly confirms deletion twice.

Portable users should extract the entire ZIP to a writable directory and run
`winTerm.exe`. Keep `portable.marker` in place to store settings in the adjacent
`data` directory.

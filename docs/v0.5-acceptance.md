# winTerm v0.5 acceptance

## Verdict

Last updated: 2026-07-17.

**Not accepted.** Visual Docking is under development. The technical spike establishes that live tab and pane transfer is viable only between windows in the current winTerm process. Cross-process transfer is not supported, and no shell restart will be used as a fallback.

`[x]` means the exact source or script check was completed in this checkout. `[ ]` means unavailable, not implemented end to end, or not manually tested.

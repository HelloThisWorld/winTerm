The native Terminal app library and Settings Model unit-test library build in
x64 Debug. All 65 tests selected by `*WinTerm*` passed. The unfiltered upstream
Settings Model suite is not a release pass: its test host terminated with
`0xC0000409` in unrelated profile/media-resource tests (144 passed, 64 failed).

winTerm application plus WinAppDriver was not available. The repository's UI
runner also retains the explicit v0.7 runtime-adapter readiness failure after
its prerequisites. Model tests are not accepted as proof of a working live
XAML pointer/overlay adapter.

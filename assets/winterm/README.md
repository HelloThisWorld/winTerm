# winTerm visual assets

This directory contains visual assets owned by winTerm. It is intentionally separate from Microsoft Terminal upstream resources.

The v0.1 artwork is an original geometric placeholder: a dark rounded square, an accent rail, and a `>_` terminal glyph. It does not use the Windows logo, Microsoft Terminal icon geometry, or Apple Terminal artwork.

Generated package assets are copied to `res/terminal/images-WinTerm` and committed so a normal application build does not require an image-generation dependency.
